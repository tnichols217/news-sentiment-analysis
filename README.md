# Sentiment Analaysis on American News


## Setting up environment

With nix: enter the development shell with `nix develop`, then fill in `.env`

### Without nix

1. Install `python`
2. Install all pip dependencies listed in the `flake.nix`
3. Set up `basedpyright` and `ruff` for formatting and linting
4. Fill in `.env`

## Running the project

From the root directory, run `python -m src.main` to run the project
